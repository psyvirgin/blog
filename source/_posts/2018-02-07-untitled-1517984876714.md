kali linux不用介绍了，hacker系统，要用起来需要适应的过程，重要的是以下方法的可选择性
```bash
$ apt-get install git
$ git clone https://github.com/LionSec/katoolin.git  && cp katoolin/katoolin.py /usr/bin/katoolin
$ chmod +x  /usr/bin/katoolin
$ katoolin
```
